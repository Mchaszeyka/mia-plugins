# Nurture Sequence Architect — Claude Cowork Plugin

Builds complete, deployable nurture sequence playbooks. Not just email copy — the full operational spec: enrollment criteria, suppression logic, sequence map, branch logic, A/B test plan, and success metrics.

**Built by:** [Mia Chaszeyka](https://miachaszeyka.com) — Marketing Operations & AI PM  
*Built from experience managing 125+ email sequences across 5 personas for CNCF Silver membership programs.*

---

## What It Does

Provide your ICP, offer, and funnel stage. Get back a deployable playbook:

- Enrollment trigger logic (HubSpot-ready)
- Suppression rules
- Sequence map (email name, day, goal, CTA, success signal)
- Per-email briefs with subject line A/B options, preview text, body structure, and branch logic
- Exit criteria
- A/B test plan with sample size guidance
- Success metrics with targets

Works for: lead nurture, trial onboarding, re-engagement, post-event, member onboarding, competitive displacement, and expansion/upsell.

---

## Installation

```bash
/plugin install https://github.com/miachaszeyka/mia-plugins/nurture-sequence-architect
```

---

## Commands

```
/nurture-sequence-architect:build   — Build a new sequence playbook from scratch
/nurture-sequence-architect:extend  — Add emails or branch logic to an existing sequence
```
